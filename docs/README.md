# Documentation Index

This folder contains the project documentation for the BCNN training flow and the RTL accelerator.

## Files

- `project_description.md` - short project summary
- `SPEC.md` - intended system requirements and expected workflow
- `STATUS.md` - current implementation status, strengths, and open risks
- `RTL Design Doc.md` - module-by-module RTL design notes migrated from `verilog/README.md`

## Suggested reading order

1. Start with `project_description.md` for the one-page overview.
2. Read `SPEC.md` for the target system and reproduction expectations.
3. Read `RTL Design Doc.md` for the current Verilog implementation details.
4. Use `STATUS.md` to understand what is implemented, what is incomplete, and what still needs validation.
