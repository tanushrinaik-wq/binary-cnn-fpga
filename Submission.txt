Project Description:

EdgeProbe is an end-to-end Edge AI project showcasing the training of a lightweight Binary Convolutional Neural Network (BCNN) in software and its deployment as a real-time hardware accelerator on an FPGA. The objective of the project is to reduce computation cost, power consumption and latency for edge devices by replacing the traditional floating-point CNN operations with efficient binary operations like XNOR and popcount.

The workflow starts with training a small BCNN student model for the Rock-Paper-Scissors dataset with Straight-Through Estimator (STE) based binarization and Knowledge Distillation from a high-accuracy MobileNetV2 teacher network. The trained binary weights are then exported to Verilog compatible memory files and implemented on an Intel/Altera Cyclone IV FPGA.

On hardware, the accelerator receives image data through SPI, converts grayscale pixels into binary form, performs two layers of binary convolution using XNOR + popcount logic, applies global average pooling and fixed-point classification, and outputs a final prediction in a streaming pipeline.
